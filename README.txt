Risky Business

java *.java compiles the program.

java Main runs the program.

Play the game as instructed on the Command Line Interface.
Rules follow those of the classic game of Risk, except for the following: 
 - Player can cash in either 0 of their cards or all of their cards
 - Each card translates to 1 army when it is cashed in
 - Every country initially has 5 armies on it
